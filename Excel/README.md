# Excel Analysis

## Overview
Analysis of 970 gym members' exercise habits and health metrics using Excel Online.

## Skills Demonstrated
- Pivot tables and slicers
- Calculated columns using IF formulas
- Data visualisation with charts
- Critical analysis of results

## Key Findings
- HIIT sessions burned the most calories on average despite shorter duration
- BMI remained consistent across all experience levels
- Workout frequency was almost identical across genders at 3.3 days per week
- Strength sessions had the longest average duration
