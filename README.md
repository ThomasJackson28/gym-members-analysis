# Gym Members Analysis

## Overview
An analysis of 970 gym members' exercise habits and health metrics 
to uncover trends in workout performance, calorie burning, and 
member experience levels.

## Dataset
- Source: Kaggle (Vala Khorasani)
- 970 rows of gym member data
- Columns include: age, gender, weight, BMI, workout type, 
  calories burned, session duration, and experience level

## Tools Used
| Tool | Purpose |
|------|---------|
| Excel | Pivot table analysis and visualisation |
| SQL | Querying and validating findings |

## Structure
- 📁 [Excel](./Excel) — pivot tables and charts
- 📁 [SQL](./SQL) — queries and results

## Key Findings
- HIIT is the most time-efficient workout for calorie burning
- BMI is consistent across all experience levels
- Workout frequency is equal across genders at 3.3 days per week
- Strength sessions have the longest average duration
- Fat percentage decreases as experience level increases
